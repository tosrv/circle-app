## About
Circle is a social media app that lets you **create your own circle**, follow friends, and share threads.  
Perfect for communities and interest groups.  

---

## Features
✨ **Key Features**:
- 🧵 **Threads**: Post and interact with threads  
- 🔍 **Search**: Find users quickly  
- 👥 **Follows**: Follow your friends and circles  
- 👤 **Profile**: Personal profile and activity feed  

---

## Tech Stack
🛠️ **Technologies used**:
- **Backend**: Node, Express, Prisma  
- **Frontend**: Vite, React, TailwindCSS, Shadcn/UI  

---

## Installation
Steps to run this project locally:

```bash
git clone https://github.com/tosrv/DW-circle-app.git
cd DW-circle-app
npm install
npm start
```
